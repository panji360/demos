Web Augmented Reality Demo - Panji Across SEA

    How to use this webAR
  1. Open the scanner web app from the QR code or link on the left with a phone's browser
  2. View the markers below on a desktop or another screen (or print them)
  3. Scan the markers using the scanner web app on your phone (requires camera access)
  4. Rotate or zoom on the 3D models using your phone's touch screen
